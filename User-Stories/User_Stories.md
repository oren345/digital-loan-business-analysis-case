# User Stories

## US-01 Loan Application Submission
As a customer,
I want to submit a loan application,
So that I can receive quick evaluation.

Acceptance Criteria:
- All mandatory fields must be completed.
- Income must be numeric.
- System must validate identity number.
- Response must be returned within 5 seconds.

As a system,
